Borderlands UI
==============

This is UI (User Interface) library for ReactJS.


Installation
------------

```
yarn add https://github.com/hezymal/borderlands-ui.git --save
```


Build
-----

for build in Yarn:
```
yarn run build
```

for build examples in Yarn:
```
yarn run build:examples
```

for run watching in Yarn: 
```
yarn run watch:examples
```
